# Mayank Gupta | Full Stack & AI Engineer Portfolio

Personal portfolio for [maynk.me](https://maynk.me), focused on full stack engineering, AI agents, automation systems, React interfaces, Python backends, FastAPI services, and LLM-powered workflows.

## Highlights

- Search-friendly metadata, canonical URL, Open Graph tags, Twitter card tags, and structured Schema.org data.
- Stable favicon and social preview assets for stronger brand visibility in Google Search and link previews.
- Project descriptions that explain the problem, architecture, stack, AI systems, backend systems, and deployment context.
- Crawlable portfolio copy reinforcing Full Stack Engineer, AI Engineer, React Developer, Python Developer, FastAPI, LangChain, LLM Systems, AI Agents, and Automation Systems.

## Stack

- React + Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js
- React Helmet Async
